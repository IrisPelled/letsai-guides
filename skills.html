מה זה בעצם Skill?

💡 הגדרה

Skill הוא קובץ הוראות שמלמד את Claude לבצע סוג מסוים של עבודה — עיצוב, כתיבה, קוד, ניתוח נתונים, ועוד — בצורה עקבית, מקצועית, ולפי הסטנדרטים שאתם בחרתם.
במילים פשוטות: אתם לא מסבירים לו שוב ושוב — אתם מלמדים אותו פעם אחת.

💭 אנלוגיה דמיינו שאתם שוכרים עובד חדש. בלי Skills — אתם מסבירים לו מחדש בכל בוקר איך לעבוד, מה הסגנון, מה חשוב ומה לא. עם Skills — העובד קיבל את הנהלים בהתחלה, וכבר יודע. זה עובד לכל תחום: עיצוב, כתיבת תוכן, פיתוח, ניתוח נתונים, ועוד.

2 דרכי התקנה

מומלץ הכי קל

ראשית, וודאו שיש לכם תיקיית Skills. כתבו לקלוד:

בדוק אם יש לי תיקיית skills ואם לא תיצור לי אחת

אחרי שיש לכם תיקייה, פשוט כתבו לקלוד:

תתקין לי את הסקיל הזה: [הדביקו כאן את הקישור של הסקיל]
💡 טיפ

אפשר לבקש להתקין כמה Skills בבת אחת — פשוט שלחו את כל הקישורים ביחד.

1. פתחו את תיקיית Skills לחצו Win+R והקלידו: %USERPROFILE%\.claude\skills
2. צרו תיקייה אם לא קיימת אם התיקייה .claude לא קיימת — צרו אותה
3. הורידו את ה-Skill קובץ .md או תיקייה מלאה
4. שמרו בתוך תיקיית skills ישירות בתוך ~/.claude/skills/
5. הפעילו מחדש את Claude Code סגרו ופתחו שיחה חדשה
⚠️ שימו לב

אם התיקייה .claude לא קיימת — בקשו מ-Claude Code ליצור אותה, זו הדרך הכי בטוחה.

שלב חובה — הפעלת ה-Skill

⚠️ חשוב!

לשים את הקובץ בתיקיית skills זה לא מספיק! צריך גם להפעיל את ה-Skill דרך הממשק של Claude Code.

1. לחצו על לשונית Customize בממשק של Claude Code, מצאו את לשונית Customize
2. בחרו ב-Skills בתפריט שנפתח, בחרו את האפשרות Skills
3. לחצו על + (הוספה) לחצו על כפתור הפלוס כדי להוסיף Skill חדש
4. העלו את קובץ ה-.md בחרו את קובץ ה-.md של ה-Skill מתיקיית skills והעלו אותו
5. חזרו על זה לכל Skill כל קובץ .md צריך להיות מועלה בנפרד
💡 טיפ

רק אחרי ההעלאה דרך Customize → Skills → + , ה-Skill באמת פעיל ו-Claude ישתמש בו.

איך לדעת שזה עבד?

✅ בדיקה

פתחו שיחה חדשה ב-Claude Code וכתבו את הסימן / (סלאש). תיפתח חלונית עם רשימת כל ה-Skills המותקנים אצלכם.

ה-Skill ייטען אוטומטית ו-Claude ישתמש בו כדי לבצע את הבקשה שלכם ברמה הכי גבוהה.

נקודת ההתחלה: ה-PRD

📋 מה זה PRD?

PRD = Product Requirements Document = מסמך שמתאר מה רוצים לבנות.
אתם כותבים מה האתר צריך לעשות — Claude מחליט איך לבנות אותו.

אחרי שנותנים PRD — Claude שואל 3 שאלות:

  1. יש לך design system או אתר reference לסגנון?
  2. יש לך לוגו, תמונות, או סרטונים?
  3. האתר בעברית?
